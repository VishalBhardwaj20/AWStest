# AWS Dashboard — Tata Nexarc

A simple Angular dashboard UI for monitoring AWS infrastructure.

## Features
- EC2 instance overview
- Storage stats
- Uptime monitoring
- Active alerts panel

## Setup
```bash
npm install
ng serve
```

Open `http://localhost:4200`

## Build
```bash
ng build
```

## Stack
- Angular 19
- SCSS
- TypeScript
